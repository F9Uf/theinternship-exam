# Weather :sparkles:

## How to run :confused:
- Clone or Download file from This repo.
- Open **"weather"** folder.
- Run this commad.
    ```
    $ python wml-json.py {fileName}.xml
    ```
- It return `{fileName}.json`.If this file name already exists, It will create new file name.
